## Installation
Install Composer

```
$ curl -sS https://getcomposer.org/installer | php
$ sudo mv composer.phar /usr/local/bin/composer
```

Craete a database and change .env accordingly

Run Migration

```
php artisan migrate
```

Run local server

```
php artisan serve
```

Visit localhost:8000
